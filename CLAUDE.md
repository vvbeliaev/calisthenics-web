# Caliathletics — Персональный бренд калистеники

## Миссия проекта

Создать **впечатляющий "вау" продукт** — сайт личного бренда по калистенике, который вызывает эмоции, вдохновляет и конвертирует посетителей в клиентов. Сайт должен быть на уровне лучших фитнес-лендингов мира, но с душой и характером калистеники.

## О бренде

**Caliathletics** — персональный бренд тренера по калистенике. Философия: "Построй силу, которая с тобой навсегда". Тренировки с собственным весом тела, без оборудования, доступные каждому.

### Ценности бренда
- **Свобода** — тренируйся где угодно, когда угодно
- **Прогрессия** — от новичка до продвинутого уровня через систему уровней
- **Естественность** — твоё тело — твой главный снаряд
- **Дисциплина** — чёткий график 4 дня тренировок / 3 дня восстановления
- **Трансформация** — реальные результаты, подтверждённые отзывами

### Целевая аудитория
- Люди 16-40 лет, которые хотят стать сильнее без тренажёрного зала
- Новички без опыта тренировок
- Те, кто не может заниматься с отягощениями (травмы, ограничения)
- Люди, ищущие индивидуальный подход с личным тренером

## Структура сайта (одностраничный лендинг)

### 1. Hero секция
- **Заголовок:** "Построй силу, которая с тобой навсегда"
- **Подзаголовки:** Индивидуальная программа / 4 дня в неделю / Отдых и восстановление
- **Описание:** Готовый план тренировок 4 дня в неделю. Без оборудования. Только ты и твой вес.
- **CTA:** "Получить программу тренировок"
- **Фон:** Видео с тренировками

### 2. Преимущества калистеники
- **Чёткий график** — Пн, Вт, Чт, Пт — сила. Ср, Сб, Вс — восстановление.
- **Устойчивый рост** — Каждая тренировка прорабатывает все тело.
- **Без оборудования** — Твоё тело — твой главный снаряд.
- **Прогрессия** — Упражнения имеют уровни сложности.

### 3. Программа тренировок
Расписание на неделю:
| День | Тип | Описание |
|------|-----|----------|
| Понедельник | PUSH (Жимовая) | Грудь, Плечи, Трицепс |
| Вторник | LEGS (Ноги) | Сила и взрывная мощь |
| Среда | REST (Отдых) | Активное восстановление |
| Четверг | PULL (Тяговая) | Спина, бицепс, предплечья |
| Пятница | CORE (Cardio) | Пресс, выносливость |
| Сб/Вс | Полный отдых | Растяжка, дыхание |

Описание платформы: видеоматериалы с техникой и разъяснением мышц, обновлённый интерфейс.

## Технологический стек

- **Framework:** Astro v6 (SSG)
- **UI Framework:** Svelte 5 (через @astrojs/svelte) — интерактивные компоненты
- **UI Kit:** shadcn-svelte (стиль "vega") — компонентная система
- **Стилизация:** Tailwind CSS v4 + tw-animate-css
- **Шрифты:** Inter Variable (основной), Bebas Neue (заголовки)
- **Иконки:** Lucide (через shadcn-svelte)
- **Язык:** TypeScript (strict)
- **Пакетный менеджер:** pnpm
- **Анимации:** CSS-анимации, intersection observer для scroll-анимаций
- **Деплой:** будет определён позже

### Установленные shadcn-svelte компоненты
accordion, avatar, badge, button, card, carousel, collapsible, dialog, form, input, label, scroll-area, separator, sheet, tabs, textarea, tooltip

### Добавление компонентов
```bash
echo "y" | pnpm dlx shadcn-svelte@latest add <component-name> --overwrite
```

### Дизайн-токены (oklch, тёмная тема)
- `--primary` = brand orange (`oklch(0.72 0.19 55)` ≈ #ff8400)
- `--accent` = brand gold (`oklch(0.88 0.17 90)` ≈ #ffd500)
- `--background` = deep dark (`oklch(0.13 0.01 175)` ≈ #06100f)
- `--brand-orange` / `--brand-gold` / `--brand-dark` — прямые brand-токены для градиентов
- Сайт всегда в dark mode (`<html class="dark">`)

## Принципы разработки

- **Performance first** — минимальный JS, оптимизированные изображения, lazy loading
- **Mobile first** — адаптивный дизайн начиная с мобильных
- **Accessibility** — семантический HTML, aria-атрибуты, контрастность
- **SEO** — мета-теги, структурированные данные, Open Graph
- **Wow-эффект** — плавные анимации при скролле, микроинтеракции, параллакс, видео-фоны
